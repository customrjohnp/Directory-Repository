How to add new emails
  1. Export your email HTML (from Emarsys preview → HTML) and save it inside /emails/.
  2. Update the EMAILS array inside this file with a new object pointing to your HTML file. Use the naming convention fields to keep things tidy.
  3. Commit & push. GitHub Pages will serve each file and this index instantly.
