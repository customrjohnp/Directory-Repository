How to add new emails (step-by-step)
  1. Export your email HTML from Emarsys preview → save into /emails/ (or your chosen folders) with a .html extension.
  2. In the EMAILS DATA – EDIT HERE block, duplicate an item and update values between quotes only.
  3. path can have spaces/&/() — the page will encode them. If you forget .html the page will add it for preview/copy/download.
